MatchMate - SwiftUI

A simple matrimonial matchmaking application built using SwiftUI following the MVVM Architecture and Repository Pattern.

Features
Fetches profiles from the RandomUser API
SwiftUI-based Match Cards
Accept / Decline functionality
Core Data local persistence
Offline support using cached data
MVVM + Repository Pattern
Async/Await networking with URLSession
Architecture
SwiftUI
MVVM

Clone the repository.
